# 供热工程复习知识点

1. 供热系统的组成
   * **热源**、**管网**、**热用户**三部分组成，连接这三部分的是**热媒**。
   
2. 蒸汽供热系统的组成，蒸汽热媒的特点，工业企业为何大多采用饱和蒸汽而不是过热蒸汽作为热媒。
   * **热源设备**、**蒸汽管网**、**蒸汽用户**和**凝结水回收系统**共同组成。
   * 蒸汽特点：水蒸汽洁净，无污染，而且水蒸汽焓值高，载热量大；凝结放热时温度基本保持不变，能满足某些需要**定温加热**的生产工艺的需要；可以根据各用热设备对热媒参数（温度、压力）的不同要求进行**逐级降压，梯级使用**；可以在系统中设置蒸汽蓄热器蓄聚蒸汽热能；蒸汽作为热媒的适用面较广。
   * 主要是**饱和蒸汽**，是由于蒸汽放热通常属于相变放热（凝结放热），其放热系数很高。因而可以减少放热面积，减少换热设备的金属耗量，降低设备费用。而过热蒸汽不会冷凝，只会产生通常的对流换热，放热系数远小于饱和蒸汽凝结放热。
   
3. 蒸汽参数
   * **蒸汽量**、**蒸汽品质**（含水率-湿度)、**压力**、**温度** 四项。
   * 蒸汽量一般要根据供热系统的热负荷进行确定
   * 锅炉出口的蒸汽湿度（含水率）一般以不大于3%为宜，不得大于5%
   * 压力与温度是一一对应的，所以，只要控制好压力即可
   * 压力选取：用户要求的最高压力加上管网压力损失，再加上0.1～0.2MPa的富裕值
   * 温度选取：用户要求的最高温度加上管网散热造成的温降，再加上5～10℃的富裕值，可采用**减温减压器**进行调节
   
4. 管网的型式：环状、枝状管网，优缺点比较。管制：单管制（常用）、双管制。
   * **枝状管网**：布置比较简单；随着距热源距离的增大，管网的主干管径可逐渐变小，因此管网的金属耗量小；基建投资少；运行管理也比较方便。**缺点**：不具备后备供热能力。当管网主干线某处发主故障时，在故障点以后的各热用户的供热都将停止。**对策**：为了在管网发生故障时，缩小事故的影响范围、迅速消除故障，在与干管相连的管路分支处，及在与分支管路（旁通管）相连接的较长的用户支管处，均应装设供启闭用的闭路阀门；或采用双管制。
   * **环状管网**：如主干线上某处出现故障，相邻的枝干线可从环路另一方向供热。所以，每一个枝干线都有双回路供热，供热的后备能力很强。**缺点**：管网投资比较大，运行管理复杂，要求的自动化程度较高。管网环状管网主要用于多热源的大型集中供热系统，所以目前在供热管网中很少采用。
   * **单管制**：各主干线、支干线和用户支线均为：单根供汽管（去）——单根凝结水管（回）
   * **双管制**：既可以提高生产性供汽的可靠性，也便于对季节性用汽进行集中调节，同时，也可减少非采暖季节管道的散热损失。缺点：双线管网的投资比单线管网的投资要高。
   
5. 上下级管网之间的连接

   * **直接连接**：下级管路只有一根或两根时——可以直接接在上级管路上。
   * **分气缸**：当供汽主干管道多于两根时——为避免上级管路上开孔过多，一般应接口处设置分汽缸下级管路均应从分汽缸中接出。
   * **热力站**：如上下级管路的热媒种类不同，或热媒参数相差较大——则宜设置热力站，在热力站中设置必要的热交换设备、凝结水回收设备或减温减压设备。

6. 蒸汽管网与热用户之间的连接方式，如管路中设备的作用。

   1. 可以或需要使用蒸汽时
      * 如与用户压力相等——则直接连接
      * 如用户压力较低——则一般要用减压阀减压后连接
      * 减压阀前后要装压力表、阀门，出故障时方便更换维修
      * 减压阀后一般要装安全阀，以防减压阀失灵，用安全阀泄压
   2. 需要使用热水或其它热媒时
      * 一般要用换热器进行换热，用蒸汽加热其它热媒
   3. 用户采用表面上加热器时
      * 一般要考虑凝结水的回收利用问题
   4. 对于热水供应
      * 即热式——加热器不能放在大容积的储水箱中，要强制流动
      * 蓄热式——加热器可在储水箱中自然对流加热

7. 季节性热负荷、常年性热负荷、最大热负荷（确定供热锅炉容量、类型、台数的基本依据）、平均热负荷（与最大热负荷之差确定备用锅炉的依据）、年总热负荷（设计贮煤场、灰场的依据）。如果系统中没有设置蓄热器，锅炉供热量应该以最大热负荷为准；如果有，则以平均热负荷为准。

   * 供暖、通风、空气调节系统的热负荷是**季节性热负荷**。 对它的大小起决定性作用的是室外温度。（室外气温对它的影响最大，由于室外气温随季节改变有很大的变化。）

   * 生活热水（主要指热水供应）和生产工艺系统用热属于**常年性热负荷**。 与气候条件关系不大，它的用热状况在全日中变化较大。（一天之中可能变化很大，但一年之中每天变化不大）

   * 供热系统的**最大热负荷**：是确定供热锅炉容量、类型、台数的基本依据。为了减少锅炉房的最大热负荷，从而降低锅炉设备的总投资。

     在组织生产时，应尽量使各车间及车间的各用热设备的高峰负荷错开

     在设计时，可采用蒸汽蓄热装置，削峰填谷，使最大热负荷接近平均热负荷

   * 供热系统的**平均热负荷**：与最大热负荷的差值是选配备用锅炉时的主要参考依据

   * 供热系统的**年总热负荷**：决定了锅炉房的年总耗煤量和年总灰渣量是设计锅炉房贮煤场、灰场的主要参考依据。

8. 凝结水回收利用的意义和一般原则。

   * **意义**：
     1. 凝结水是水（节约水资源）
     2. 凝结水是良好的锅炉补给水（降低设备投资和运行费用）
     3. 凝结水中含有大量的热量，通常约占蒸汽本身热量的12％～15％，如果包括疏水器的漏汽、窜汽，这一比例则可达 20～50％ 或更高（节约热能，降低燃料费用，降低热负荷，减少设备投资）
     4. 凝结水回收存在问题会造成供热不良
   * **原则**：
     1. 应尽量采用蒸汽间接加热的方式（以提高凝结水的回收利用率）
     2. 凝结水应尽可能回收（除被加热介质有毒（如氰化物液体等）或有强烈腐蚀性的溶液外）
     3. 应该充分利用凝结水的热量（梯级利用、降温排放，就地排放的凝结水温度，不得高于40℃）
     4. 回收的凝结水应符合锅炉给水水质标准（达不到标准时，应进行水质处理）
     5. 凝结水回收系统应尽可能采用闭式系统（防止空气渗入系统造成管路和设备的腐蚀）

9. 凝水回收系统分类，重力回水、余压回水和加压回水系统的特点。

   * **分类**：
     1. 按其是否与大气相通，可分为：开式凝结水回收系统、闭式凝结水回收系统
     2. 驱使凝水流动的动力不同，可分为：重力回水、余压回水、加压回水、混合回水
     3. 按凝水的流动流动方式，可分为：单相流、两相流
   * **重力回水**：供热系统规模较小，室外地形较平坦的厂区。凝水管路多采用地下敷设，锅炉房（或凝水分站）应处于地势较低洼的地方。
   * **余压回水**：所有余压回水系统都是满管流。余压回水系统非常简单，热用户处只有疏水器。室外凝水管可随地形向上或向下倾斜，可以架空或地下敷设。闭式余压回水系统的**二次蒸汽**可在锅炉房或凝水分站中集中利用。缺点：高低压回水互相干扰，低压设备难以回水。漏汽、凝结水降压汽化，形成**水击**。流体密度大幅下降，需要管径较粗。
   * **加压回水**：以大用户为中心，设置凝水箱收集邻近几个用户流来的凝结水，在排出或利用水箱内的二次蒸汽后，用凝水泵或其它加压装置将纯凝水输送回热源的总凝水箱。适用于用热设备较多，用户较为分散且距热源较远的较大的蒸汽供热系统。通常采用离心式水泵作为凝结水加压泵。在供热系统中使用离心泵时必须引起注意的一个重要问题是**离心泵的“汽蚀”现象**。

10. 水击与汽蚀定义及其防治措施：降温升压（升压的途径减小吸水管道的流动阻力、防止发生局部过热、控制离心泵的安装高度、在离心泵之前设置前置泵等)。

    * **水击**：汽水两相流动时，如果汽水混合物流速较大，则高速流动的水塞，像众多的锤子一样，频繁地在管道转弯处或阀件处敲击，严重的水击能破坏管件和设备。所以，一般来说，无论是开式系统，还是闭式系统，都要尽可能在用户处以及在沿程途中

      * 将蒸汽排出——开式系统
      * 将饱和凝结水进行降压汽化——闭式系统
      * 或者，使凝结水继续放热降温，回形管处**不保温** 

    * **汽蚀**：在压力最低的地方，液体的绝对压力 $\le$ 液体相应温度下的饱和压力，液体将发生汽化，产生气泡。气泡进入高压区破裂，从而引发周围液体高频碰撞的过程。

      防治措施：降温、升压

      1. **降温**：让凝结水自然降温或与冷却水混合冷却，再送入泵中，这种方式会损失大量热能或电能，加冷水混合还会加重回水管网的负担，设计时需加粗回水管径。因此，一般**不推荐**使用这一方案。

      2. **升压**：

         * 尽量减少吸水管道的流动损失

         * 防止发生局部过热

         * 控制离心泵的安装高度

         * 在离心泵之前设置前置泵★（低转速前置离心泵、前置喷射泵）

11. 凝结水回收系统设计具体应注意的问题，如系统腐蚀问题，相应策略。

    * **回水动力的大小**：加压（不经济）、有余压可延长管长 $l$ 加粗管径 $d$ 
    * **凝结水系统含汽问题**：回水系统一定不能含汽
    * **二次蒸汽如何利用**：采暖系统、低压用热设备、加热锅炉软化水、生活热水供应或生产工艺用水
    * **系统腐蚀**：系统中混有空气——化学腐蚀与电化学腐蚀（氧气腐蚀）；管道中汽水分层流动（非满管流动）（疲劳腐蚀）。为了减少腐蚀：采用闭式系统；尽量排除蒸汽；管路设计时，尽量采用满管回水（回形管）
    * **防止凝结水泵的汽蚀**
    * **避免各用户回水相互干扰**：对蒸汽压力相差较大的用户，其凝水可专管输送；如采用合管输送，应仔细计算各用户系统的流量和压力损失；精心调节、维护和管理疏水器

12. ★疏水器的作用、性能要求及其分类，机械型（疏水及时，良好的水封不漏汽、能排饱和凝结水、不能排空气）、热动力型（存在原理性漏汽，漏气率最高、不能排除饱和凝结水）、热静力型疏水器（具有良好的排空气性能，温度敏感元件需有一定的滞后时间才能启闭阀门，不能排出饱和温度的凝结水，并需要较大的过冷度）的原理及其优缺点。

    * **作用**：**阻汽排水**。排水——能自动而迅速地排出用汽设备及供汽管道中的凝水；阻汽——阻止蒸汽进入凝水回收系统。
    * **性能要求**：排水量大、排水及时、漏汽量小、能排除空气等不凝性气体、能承受一定的背压、结构简单、重量轻、运行可靠，内部部件不易磨损并且易于拆修
    * **分类**：
      * **机械型**（依靠蒸汽和凝水的密度差驱动）：疏水及时，良好的水封不漏汽、能排饱和凝结水、不能排空气
      * **热静力型**（依靠蒸汽和凝水的温度差驱动）：存在原理性漏汽，漏气率最高、不能排除饱和凝结水
      * **热动力型**（依靠蒸汽和凝水的流动特性差驱动）：具有良好的排空气性能，温度敏感元件需有一定的滞后时间才能启闭阀门，不能排出饱和温度的凝结水，并需要较大的过冷度

13. ★疏水器的安装、管路配置与检测方法。

    <img src="3.%E5%87%9D%E7%BB%93%E6%B0%B4%E5%9B%9E%E6%94%B6%E5%88%A9%E7%94%A8.assets/image-20230310153711076.png" alt="image-20230310153711076" style="zoom:50%;" />

    * 应靠近用汽设备安装，圆盘式前一般有一段冷却管。应安装在设备最低排水点以下。一般应每台设备安装一只和 $2～3$ 只疏水器。原则上不应几台设备共用一只，以免回水相互干扰。疏水器通常多为水平安装（即安装在水平管道上）

    * 建议配置：<img src="%E6%9C%9F%E7%BB%88%E8%80%83%E8%AF%95.assets/image-20230320081709313.png" alt="image-20230320081709313" style="zoom:30%;" />

    * 前后需设置**闭路阀门**——供疏水器检修时截断用

    * 疏水器后应设置**检查管**——用以检查疏水器的工作情况是否漏汽

    * **止回阀**——某些情况下，管路中还需装置止回阀，例如：

      疏水器后凝水向上提升至架空敷设的凝水主干管，或与背压较高的凝水管并联时，为了隔断疏水器后过高的背压、避免凝水向设备倒灌都要设置止回阀。

    * 疏水器前应设置**冲洗管**——用以冲洗管路中的污物以及排放空气

      当管路及设备安装及检修完毕，投入运行以前应通过冲洗管尽可能将系统中的脏物冲洗干净

    * 疏水器前端应设**过滤器**——过滤凝水中含有的渣垢杂质

      过滤器中置有金属丝编织成的滤网或开有许多小孔的金属滤板。

      疏水器本身带有过滤网时，可不设过滤器。

      平时运行一段时间后也要将滤网取出清洗或加以更换。

    * 旁通管——主要作用是

      * 在开始运行时排除大量凝水和空气
      * 供汽不允许中断的设备可以在设备不停机情况下检修疏水器

    * 检测和判断方法：

      #### 1、目测法

      打开**检查管**，让凝水排入大气

      * 如从检查管喷出一丝丝白色的蒸汽，则说明**不漏汽或漏汽不严重**（白色蒸汽为凝水排向大气时由于降压而闪蒸出的二次蒸汽）
      * 如喷出的是无色透明或呈浅蓝色透明的高速汽流，则为新鲜蒸汽，说明**疏水器漏汽严重**；

      #### 2、测温法

      用**接触式点温计**测量疏水器前后接管的温度。也可在前后接管上洒一些水，观察其蒸发的快慢，来定性地观察其温度高低

      * 如前后接管温差较大，说明疏水器**工作正常** 
      * 如温度很接近，则说明有**严重的漏汽** 

      #### 3、听诊法

      用**听诊器或金属棒**探听疏水器开闭动作的规律，从而判断其工作是否正常

      * 一般情况下，若疏水器周期动作，排水爽快，阻汽时间比排水时间略长（阻汽时间约为排水时间的一至四倍），动作周期约半分钟左右时，则说明**运行正常**。
      * 若疏水器不排水、不漏汽、久久没有动作，用汽设备的加热远远不能达到要求，则说明疏水器被**完全堵塞**。可能的原因是：
        * 前面的阀门被误关闭
        * 管路、设备或过滤器被堵死
        * 疏水器阀孔被堵塞
        * 吊桶过轻不能落下
        * 浮球被压坏不能浮起等。
      * 若疏水器持续排水，久排不闭，这表明：
        * 用汽设备还在升温阶段
        * 否则就是疏水器选用得过小。

16. 蒸汽供热系统热负荷变化的原因、危害、特征以及解决方案。

    * **热负荷变化原因**：
      1. 有的用汽设备是周期性运行的
      2. 生产过程不连续，生产班次不同
      3. 有的用汽设备是不规则地间歇运行的
      4. 其它不可预测的用汽
    * **危害**：
      1. 增加备用锅炉的台数，使供热系统的投资增加。
      2. 会降低锅炉运行的经济性
      3. 会引起锅炉运行上的不安全
      4. 影响企业产品的产量和质量
      5. 司炉人员的劳动强度大大增强
      6. 燃烧的不完全，还会冒黑烟，污染环境
    * **特征**：
      1. 有的变化平缓，有的变化剧烈
      2. 有的变化呈一定的周期性，有的变化则没有明显的规律
      3. 有的是季节性变化，有的是以几小时、一两天等为变化周期
    * **解决方案**：
      1. 对于季节性的变化（如采暖通风用热）一般可以通过**增减锅炉运行的台数**来进行调节
      2. 对于短时间（如几小时、一两天等）内的负荷变化可以
         * 通过**生产调度**，将主要用汽设备的用汽高峰错开
         * 如果不能错开，或错开后波动仍然很大，则可以考虑**设置蒸汽蓄热器** 
      3. 当用户用汽量小于锅炉供汽量时，锅炉多余的蒸汽进入蓄热器中——使水的温度和压力升高，水位上升
      4. 当用户用汽量大于锅炉供汽量时，蓄热器降压排汽，弥补锅炉供汽量的不足

17. 蒸汽蓄热器的作用有哪些。

    * 保证锅炉在稳定的平均负荷下安全经济地运行，提高锅炉效率，节约燃料，减少投资，减轻劳动强度，减少对大气的污染，并有利于提高产品产量和质量；
    * 当锅炉发生事故突然停炉时，蒸汽用户可有较多时间作应急处理；
    * 当工厂在夜间或休息日要用少量蒸汽时，也可以停运锅炉，而由蓄热器来供汽；
    * 有些热源（如冶金企业的余热锅炉、城市垃圾焚烧炉、太阳能集热器等）产汽量不稳定、不连续，如用蓄热器汇集间歇产生的蒸汽，则可向用户稳定连续地供应蒸汽；
    * 对于热电（动）联产系统，设置蓄热器，可以平衡波动的热负荷，使汽轮机稳定运行。
